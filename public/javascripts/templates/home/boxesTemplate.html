• Login
    In order to use this App you would need a Pivotal Tracker Account to fetch all your projects. Otherwise, you should create one.
  • Choose a Project
    Once you have logged in, you will be able to navigate between Pivotal Tracker projects.
  • Create or Select a Game
    After select a project, you can create or select games. This is another name for the iteration occurred during a sprint planning meeting. Here you can assign unestimated stories to a game.
  • Play
    Wait for all the participants, start the game and PLAY!
